# GATEKEEP | Systematic Security Management System

GATEKEEP is a web application designed to streamline community security and resident management. It features a dual-interface system for security personnel and residents.

## ‚ú® Key Features
- **Security Console:** Real-time visitor registration, gate activity logs and approval request system.
- **Resident Dashboard:** Visitor approval notifications, community notice board, amenity bookings (Turf, Gym, etc.), and grievance reporting.
- **Modern UI:** Built with Tailwind CSS and Inter typography for a clean professional look.
- **Local Persistence:** Uses Browser LocalStorage to sync data between the Security and Resident modules without a complex backend.
